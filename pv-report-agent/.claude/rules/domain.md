## WHOART SOC 코드 → 기관계대분류 매핑

WHO-ART System Organ Class (SOC) 코드를 한글 기관계대분류 명칭으로 변환.
EVENT.txt의 WHOART_ARRN 앞 2자리가 SOC 구분자.

```python
WHOART_SOC_MAP = {

    "99": "기타",

# WHOART_ARRN 전체 코드 → SOC 매핑 함수
def get_soc_name(whoart_arrn: str) -> str:
    if not whoart_arrn or str(whoart_arrn).strip() == '':
        return "기타"
    prefix = str(whoart_arrn).strip()[:2].zfill(2)
    return WHOART_SOC_MAP.get(prefix, f"[미확인코드: {whoart_arrn}]")
```

## 통계 텍스트 생성 규칙

### (ㄱ) 보고 건수 텍스트
```
{제품명}에 관련된 보고건수는 총 {N}건이었으며,
이 중 신속보고는 {신속}건이었다.
시험/연구에서보고가 {시험}건, 자발적보고가 {자발}건, 기타 {기타}건이었다.
```

### (ㄴ) 인구학적 텍스트
```
보고된 인원 {N}명 중 남성은 {남}명({남비율}%)이었으며,
여성은 {여}명({여비율}%)이었다.
연령은 {연령대별 기술}이었다.
```

### (ㄷ) 이상사례 발현 현황 텍스트
```
{N}명의 환자에게 발생한 이상사례는 인과성 평가와 관계없이 총 {건수}건이었다.
주요 이상사례는 {top1} {top1건수}건({top1비율}%),
{top2} {top2건수}건({top2비율}%),
{top3} {top3건수}건({top3비율}%)이었다.
```

## 검토 필요 마커 트리거 조건

다음 경우 `[검토필요: {사유}]` 마커를 빨간 텍스트로 삽입:

1. ASSESSMENT.txt 미제공 → `[검토필요: 인과성 평가 없음]`
2. 코드집에 없는 WHOART 코드 → `[검토필요: 미확인 SOC 코드 {코드}]`
3. ADR_RESULT_CODE 미확인 → `[검토필요: 미확인 결과 코드 {코드}]`
4. 이상사례명이 비어있음 → `[검토필요: 이상사례명 없음]`

## 데이터 품질 체크

```
1. REPRT_CHANGE_CD == 1 제거 후 건수 로그 출력
2. KAERS_NO 기준 Left Join 후 고아 레코드 경고
3. 최종 df 건수 == Line Listing 행수 검증
4. 날짜 형식: YYYYMMDD 문자열 → YYYY-MM-DD 표시용 변환
```
